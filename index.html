Kcalculation Prototype

This project was made for my internship. The premise was building a mock-up of an already existing website, containing the basic functionality of the aforementioned product, but made from the Vue framework, with a backend system of my own choice, so that a comparison between the two solutions could be made. The frontend application is a small Vue prototype being able to call a REST API and display the returning data. The backend is made up by a Node.js server application and a mySQL database.
